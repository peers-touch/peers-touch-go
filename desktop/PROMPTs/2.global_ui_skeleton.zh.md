# 全局UI架构骨架

> 聚焦 “三栏骨架的功能划分、尺寸约束、场景联动逻辑”，不涉及具体组件样式，仅定义 UI 的 “结构性规则”

## 核心定位

基于阶段 1“桌面端优先、固定三栏” 的全局定位，本阶段仅定义 UI 的 “纯骨架结构”—— 明确三栏的尺寸、比例、基础交互规则，不涉及骨架内的具体内容（如入口名称、展示元素等），仅聚焦 “容器本身的规则”。

## 整体骨架结构（唯一核心：固定三栏式）

* 布局类型：左侧固定栏 + 中间主栏 + 右侧可折叠栏，三栏横向排列，全屏占比（100% 宽 ×100% 高）。
* 适配宽度范围：1200px ~ 2560px（最小适配宽度 1200px，低于此宽度不让继续缩放；高于 2560px 时三栏居中，两侧留空 #1E1E1E 背景，注意不同主流系统屏幕的单位像素密度）。
* 三栏比例与尺寸约束：
    * 左侧栏：宽度占比 20%，固定最小宽度 240px、最大宽度 320px（适配时在 240-320px 间弹性变化，确保比例协调）。
    * 中间栏：宽度占比 60%，无最小 / 最大宽度限制（随屏幕宽度按比例伸缩，是视觉核心区）。
    * 右侧栏：宽度占比 20%，固定最小宽度 240px、最大宽度 320px；支持折叠（折叠后宽度收缩至 48px，仅保留展开按钮）。

* 高度约束：三栏均为 100% 全屏高度（从屏幕顶部到底部），各栏内部独立滚动（全局无整体滚动，仅单栏内部滚动）。
* 系统边框：暂时使用系统边框

## 三栏核心定位与基础结构（仅定义 “容器规则”）

### 左侧栏：功能入口栏

* 核心定位：全局功能入口容器 + 全局状态展示容器（不承载具体内容，仅提供 “跳转 / 切换” 入口）。
* 基础结构（从上到下，仅定义区域划分，不定义内容）：
* 顶部区域：固定高度（100-120px），用于全局状态展示（如用户身份、系统连接状态等）。
* 中间区域：自适应高度（占左侧栏 60%-70%），用于垂直排列的 “功能入口组”（数量不限，每个入口为独立可点击块）。
* 底部区域：固定高度（80-100px），用于全局辅助入口（如设置、切换模式等）。
* 基础交互规则：
    * 点击中间区域的 “功能入口”，仅触发 “中间栏 + 右侧栏的内容容器切换”（无页面跳转，三栏框架始终不变）。
    * 左侧栏整体不滚动（顶部 / 中间 / 底部区域高度固定，中间入口组超出高度时，内部独立滚动）。

### 中间栏：主内容展示栏

* 核心定位：视觉焦点容器，承载当前选中功能的核心内容（是用户主要操作与浏览区域）。
* 基础结构（从上到下，仅定义区域划分，不定义内容）：
* 顶部栏：固定高度（56-64px），用于当前内容的 “标题展示”+“核心操作按钮”（如新建、刷新等）。
* 内容区：自适应高度（占中间栏 75%-85%），用于承载核心内容（仅定义为 “可滚动的内容容器”）。
* 底部栏：固定高度（48-64px），用于当前内容的 “快捷操作容器”（如输入框、发布按钮等）。
* 基础交互规则：
    * 内容区支持垂直滚动（滚动条样式统一为 “细条型”，不悬浮时隐藏，hover 时显示）。
    * 切换功能入口时，内容区仅 “容器刷新”（保留滚动容器的结构，不改变栏高、栏宽）。

### 右侧栏：辅助信息栏

* 核心定位：辅助容器，承载与中间栏当前内容相关的辅助信息（非核心操作区，可折叠）。
* 基础结构（从上到下，仅定义区域划分，不定义内容）：
* 顶部控制区：固定高度（56-64px），仅包含 “折叠 / 展开” 按钮（居右）。
* 辅助内容区：自适应高度（占右侧栏 85%-90%），用于承载辅助信息（仅定义为 “可滚动的内容容器”）。
* 基础结构（从上到下，仅定义区域划分，不定义内容）：
    * 顶部控制区：固定高度（56-64px），仅包含 “折叠 / 展开” 按钮（居右）。
    * 辅助内容区：自适应高度（占右侧栏 85%-90%），用于承载辅助信息（仅定义为 “可滚动的内容容器”）。
* 基础交互规则：
    * 折叠 / 展开：点击顶部按钮，右侧栏从 “20% 宽” 收缩至 “48px 宽”（仅保留展开按钮），再点击恢复；收缩时不影响左 / 中栏尺寸。
    * 辅助内容区支持独立垂直滚动（与中间栏滚动互不干扰）。

## 骨架级特殊状态处理（仅定义 “容器级表现”，不涉及内容）

* 加载状态：中间栏 / 右侧栏的 “内容容器” 内，显示与容器尺寸一致的 “灰色占位骨架”（无动画、无文字，仅保留容器轮廓）。
* 空状态：中间栏 / 右侧栏的 “内容容器” 无数据时，显示 “居中占位区”（仅保留容器轮廓，无插图 / 文字）。
* 错误状态：系统异常时，中间栏顶部显示 “通栏占位区”（用于承载错误提示，不改变三栏结构）。
